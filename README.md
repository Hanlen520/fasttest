## 框架介绍

`OKTest`在`macaca`的基础上做了一层关键字的封装与解析。同时支持`Android`、`iOS`两个平台，支持`自定义关键字模块`、`自定义函数`等。目的在于保证执行效率的同时更加灵活的简化用例描述。
